## Balatro Greenmod
Green up your cards, and utilize strong men to enhance the greenousity of your deck.(Note! several strong men yet to be implemented).  
This mod is my first one and also in a very early stage so it has yet to become good.

## How to use
Install steamodded: https://github.com/Steamodded/smods  
click the code button then choose download zip to download this mod.  
Unzip to a folder, it won't work if you put the loose files straight in your mods folder.  
Then put the folder in your mods folder.
